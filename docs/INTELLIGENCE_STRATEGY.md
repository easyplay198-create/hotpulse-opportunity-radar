# HotPulse 情报层战略

## 一、为什么需要出海企业情报站
只做 MVP 前验证工具会低频。用户不会每天为了看榜单回来，但用户会因为市场变化、机会变化、风险变化、案例变化回来。

因此，情报站负责解决使用频率陷阱。它是 HotPulse 的外层入口，负责让用户每天 / 每周持续打开；MVP 前验证系统是内层，负责承接决策和付费转化。

## 二、今日出海机会简报是什么
今日出海机会简报是首页第一屏模块，用来让用户每天快速知道：
- 今天有什么新机会
- 哪些方向正在升温
- 哪些机会值得验证
- 哪些风险正在上升
- 有哪些案例或商业模式值得借鉴

## 三、内容栏目
- 今日新增
- 持续升温
- 值得验证
- 风险上升
- 案例 / 模式观察

## 四、内容优先级
### P0
- AI 工具
- AI 应用
- SaaS 工具

- 效率工具
- Agent 工作流
- AI 内容工具

- 短剧

- 虚拟商品

### P2

- 平台型业务
- 泛商业模式观察

## 五、情报来源类型
1. 产品信号：App Store / Google Play / Product Hunt
2. 社区信号：HN / Reddit / Indie Hackers
3. 开发者信号：GitHub
4. 趋势信号：Google Trends / NewsAPI
5. 案例信号：出海成功案例、融资、增长报道
6. 资源信号：支付文档、本地化、CPI、上架政策

## 六、信号分类
- 资讯信号：值得知道
- 机会信号：值得观察
- 验证机会：值得行动
- 风险信号：需要警惕
- 案例信号：值得借鉴

## 七、资讯如何转译成机会
每条内容都要尽量回答：
- 这条信号说明什么
- 和哪个市场有关
- 和哪类企业有关
- 潜在机会是什么
- 风险是什么
- 可以怎么验证
- 证据来源是什么

资讯、案例、趋势和市场信号不是机会本体。HotPulse 的价值在于把这些内容转译成可追溯、可解释、可验证的机会假设。

## 八、长期目标
- 短期可以使用手工 seed / mock 验证内容结构。
- 中期接入更多数据源。
- 长期目标是自动抓取 + 算法排序 + 人工校验。
- 最终形成持续更新的出海机会情报与决策系统。
